The Shift

一人公司是叙事，
一人部门才是趋势。

硅谷反复在讲「AI 时代第一家独角兽只有一个员工」。这套叙事在 demo 层面成立。生产级业务里，Agent 时代真正稀缺的是决策权。

跨域决策（战略、法务、财务、产品、市场、人事）超出个人的认知边界与责任承担能力，必须由不同的人分担。域内执行可以由 1 个人 + 一组 Agent 完成，原本 10–20 人的部门规模被压缩到「一人部门」。

Comparison

跨域不可信，域内可压缩

一人公司 · 伪命题

跨域决策无法压缩

  • 决策范围：全公司跨域
  • 认知带宽：超出个人上限
  • 法律责任：全公司行为
  • Agent 跨域协调不可信
  • 失败成本：公司死
一人部门 · 真实趋势

单域可由一人主理

  • 决策范围：单一职能
  • 认知带宽：单域可覆盖
  • 法律责任：部门动作
  • Agent 域内执行、检索、起草
  • 失败成本：部门可纠正

Pricing Paradigm

Usage 模式：与 Agent 主动性正向耦合

Seat 模式

收入受员工数限制

  • 每加一个 Agent 都触发采购摩擦
  • 与 Agent 主动性弱相关
  • 容易被当成又一个 SaaS seat
  • 收入天花板：员工数 × 单价
Hive Usage + License

随自动化深度增长

  • 钱包 / credits 承接持续消耗
  • Trigger / Heartbeat / Dream 都计量
  • 更接近云资源 + 工作流自动化
  • 无收入天花板
