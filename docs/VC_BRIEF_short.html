Token Economy · Pricing Paradigm

三层收入 · 与 Agent 主动性正向耦合

Three Revenue Streams
01
Enterprise License
按公司 / 部门 / 私有化部署年付。多租户 · 权限 · 审计 · SSO · 合规——稳定盘，毛利 90%+
02
Usage Credits
按模型调用、工具调用、自动触发消耗。Agent 工作越多，收入与客户价值同步——复利复合
03
Workflow Packs
行业 pack / 高级工具 / 深度集成按调用扣费。复用客户已沉淀的工具与配置，毛利 70–85%
Seat 模式 · 收入受员工数限制
  • 每加一个 Agent 都触发采购摩擦
  • 容易被当成又一个 SaaS seat
  • 收入天花板：员工数 × 单价
Hive Usage + License · 随自动化深度增长
  • 钱包 / credits 承接持续消耗
  • Trigger / Heartbeat / Dream 都计量
  • 无收入天花板
